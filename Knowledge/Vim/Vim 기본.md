### 종료 옵션
`:q` 종료
`:q!` 강제종료
`:w` 저장
`:w!` 강제저장
`:wq` 저장하고 종료
`:wq!` 강제 저장하고 종료

### 입력모드
`i` insert(커서 앞)
`a` append (커서 뒤)
`I` 문장 시작
`A` 문장 끝

### 네비게이션
`0` 문장 앞으로 이동
`$` 문장 뒤로 이동
`w` word 단어
`b` backward 단어
`3w` 3단어씩 이동
`H` 화면 위
`M` 화면 중간
`L` 화면 끝
`gg` 파일 앞
`G` 파일 끝
`20G` 20번째 줄로 이동
`ctrl + u` 위로 스크롤링
`ctrl + d` 아래로 스크롤링
`{` 문단 시작
`}` 문단 끝

### 명령어
`x` 커서 아래 글자 삭제
`dd` 문장 삭제
`yy` 문장 복사
`p` 붙여넣기
`*p*` 클립보드 붙여넣기
